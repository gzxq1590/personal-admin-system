# 起動・停止手順書 (Startup & Shutdown Guide) - Pj01

本システムを利用するために、バックエンド（APIサーバー）とフロントエンド（Web画面）の両方を起動する必要があります。以下の手順に従ってください。

## 1. バックエンド (FastAPI) の起動
1. ターミナルで `Pj01/backend` ディレクトリへ移動します。
2. 以下のコマンドを実行して仮想環境を有効化し、サーバーを起動します。

```powershell
# 仮想環境の有効化
.\venv\Scripts\Activate.ps1

# サーバーの起動 (ポート 8000)
python -m uvicorn app.main:app --reload --port 8000 --host 0.0.0.0
```

- 起動後、ブラウザで `http://localhost:8000/docs` にアクセスすると、APIドキュメントが確認できます。

---

## 2. フロントエンド (Vue.js) の起動
1. 別のターミナルを開き、`Pj01/frontend` ディレクトリへ移動します。
2. 以下のコマンドを実行して開発サーバーを起動します。

```powershell
# 開発サーバーの起動 (ポート 5173)
npm run dev
```

---

## 3. アプリケーションへのアクセス
両方のサーバーが起動したら、ブラウザで以下のURLを開いてください。

- **URL**: `http://localhost:5173`
  - ※ 既にポートが使用中の場合、ターミナルの表示に従い `http://localhost:5174` 等にアクセスしてください。
  - ※ アクセスできない場合は `http://127.0.0.1:5173` (または 5174) を試してください。

画面が表示されれば準備完了です。操作方法は「操作手順書 (`operation_manual.md`)」を参照してください。

---

## 4. 停止手順
作業を終了する際は、各ターミナルで以下の操作を行ってください。

1. **サーバーの停止**:
   - ターミナル上で `Ctrl + C` を押します。
   - 「バッチ ジョブを終了しますか (Y/N)?」と表示された場合は `y` を入力して `Enter` を押します。

2. **仮想環境の無効化 (バックエンドのみ)**:
   - バックエンドのターミナルで `deactivate` と入力します。
