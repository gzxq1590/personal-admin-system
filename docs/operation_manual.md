# 操作手順書 (Operation Manual) - Pj01

本システムは3つの主要機能（ダッシュボード、ドキュメント管理、タスク管理）で構成されています。画面上部のナビゲーションメニューから各機能へ切り替えが可能です。

## 1. ダッシュボード (Dashboard)
日々の事務状況を一覧で確認し、予定を管理する画面です。

### 統計情報の確認
- 画面上部のカードに「未着手タスク数」「ドキュメント数」「今月の経費合計」がリアルタイムで表示されます。

### 予定の管理
- **追加**: 「予定追加」ボタンを押し、内容と日時を入力して「追加」を押します。
- **削除**: 予定リストの右側にある「×」ボタンを押します。

---

## 2. ドキュメント管理 (Documents)
領収書や契約書などの書類（メタデータ）を管理する画面です。

### 書類の登録
- 画面右上の「新規登録」ボタンを押し、ファイル名、種類、タグ/メモを入力して「保存」を押します。

### 編集と削除
- **編集**: 表の右端にある「✏️」ボタンを押して内容を修正し、「保存」を押します。
- **削除**: 表の右端にある「🗑️」ボタンを押し、確認ダイアログで「OK」を選択します。

---

## 3. タスク・工程管理 (Tasks)
事務手続きの進捗をカンバン形式で管理する画面です。

### タスクの追加
- 画面右上の「新規タスク追加」ボタンを押し、タイトルと詳細を入力して「保存」を押します。

### 進捗状況の変更
- 各タスクカードの下部にある「セレクトボックス」を切り替える（未着手、進行中、完了）と、タスクが自動的に対応する列へ移動します。

### 編集と削除
- **編集**: タスクカード右上の「✏️」ボタンを押します。
- **削除**: タスクカード右上の「🗑️」ボタンを押します。
